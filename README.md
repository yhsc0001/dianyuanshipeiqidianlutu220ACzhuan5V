## 电路原理

1. **输入端**：电路的输入端接收220V的交流电。
2. **整流**：通过整流电路，将220V的交流电转换为直流电。
3. **高频转换**：使用特定的芯片将直流电转换为高频交流电。
4. **变压**：高频交流电通过变压器，将其转换为接近5V的直流电。
5. **稳压**：最后，通过稳压电路，确保输出电压稳定在5V。
